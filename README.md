# 🤖 Gmail & Google Calendar AI Agent

## 📌 תיאור הפרויקט

פרויקט זה פותח במסגרת קורס **יישומי בינה מלאכותית בניהול**.

מטרת הפרויקט היא לפתח סוכן AI המבצע אוטומציה של טיפול בהזמנות לפגישות
באמצעות שילוב בין Gmail API ו-Google Calendar API.

------------------------------------------------------------------------

# 🎯 מטרות

-   התחברות מאובטחת באמצעות OAuth 2.0
-   קריאת הודעות Gmail
-   זיהוי בקשות לפגישה
-   חילוץ תאריך, שעה, מיקום ומשתתפים
-   בדיקת זמינות ביומן
-   יצירת אירוע כאשר היומן פנוי
-   יצירת טיוטת תשובה כאשר היומן תפוס

------------------------------------------------------------------------

# ⚙️ טכנולוגיות

-   Python
-   Gmail API
-   Google Calendar API
-   Google OAuth 2.0
-   Google Cloud Platform
-   uv
