# 🤖 Gmail & Google Calendar AI Agent

פרויקט במסגרת הקורס **יישומי בינה מלאכותית בניהול**.

---

# 📌 תיאור הפרויקט

מערכת חכמה המתחברת ל־**Gmail API** ול־**Google Calendar API**.

המערכת מזהה הודעות המכילות בקשות לפגישה, מחלצת באופן אוטומטי את פרטי הפגישה, בודקת את זמינות היומן ומבצעת אחת מהפעולות הבאות:

- יצירת אירוע חדש ביומן.
- יצירת טיוטת תשובה במקרה של התנגשות ביומן.

---

# 🎯 מטרות הפרויקט

✅ התחברות מאובטחת באמצעות OAuth 2.0

✅ קריאת הודעות Gmail

✅ בדיקת זמינות ביומן

✅ יצירת אירוע ביומן Google Calendar

✅ יצירת טיוטת תשובה כאשר היומן תפוס
